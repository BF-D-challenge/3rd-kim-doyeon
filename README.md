# 모디 (Modi)

> 초대장이 곧 기대감이 되는 앱. 카톡 단톡방 공지 + 구글폼으로 흩어진 모임 초대를,
> 보내는 순간 "나 센스있다" 싶은 초대 페이지 하나로.

BF.D Vibe Coding Challenge 4주차 프로젝트 · 코드네임 **모디(Modi)** · 웹앱 우선.

## 📄 문서

| 영역 | 기술 |
|------|------|
| 프론트/백 | Next.js 14 (App Router) + Tailwind CSS |
